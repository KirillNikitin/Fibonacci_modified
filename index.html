Definition: Modified Fibonacci Number (MFN) - MFN(0) = 0, MFN(1) = 1, MFN(n) = MFN(n - 2) + MFN(n - 1) / n
Try to enter any number in a form below and click "Count up".
Example values (For testing purposes):
MFN(0) = 0
MFN(1) = 1
MFN(2) = 0.5
MFN(3) = 1.1666666666666667
MFN(4) = 0.7916666666666667
...
MFN(10) = 1.3564608134920637
If you click "fill LowNumbersList" then you'll see the list of numbers with 50 following Modified Fibonacci Numbers starting from 10000000th element.
